# Lab 2 - Swagger and Postman

## Objective

Learn how to document and test ASP.NET Core Web APIs using Swagger UI and Postman.

---

## Technologies Used

- ASP.NET Core Web API
- Swagger (OpenAPI)
- Postman

---

## Steps Performed

1. Created and executed an ASP.NET Core Web API.
2. Enabled Swagger documentation.
3. Accessed the API documentation through Swagger UI.
4. Installed and configured Postman.
5. Tested the API endpoints using HTTP GET requests.
6. Verified successful responses from the API.
