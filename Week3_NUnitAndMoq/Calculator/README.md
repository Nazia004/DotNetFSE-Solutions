# Calculator Tests

This project contains NUnit test cases for the `CalcLibrary` project.

## Features Tested

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero exception
- AllClear()
- GetResult property
- Parameterized tests using TestCase

## Technologies

- C#
- .NET 10
- NUnit
- NUnit3TestAdapter
- Microsoft.NET.Test.Sdk

## Run

```bash
dotnet test
```

## Result

All test cases passed successfully.
