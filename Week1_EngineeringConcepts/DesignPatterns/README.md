# Design Patterns

This folder contains implementations of the Gang of Four (GoF) Design Patterns in C#.

## Creational

- LoggerSystem (Singleton)
- DocumentFactory (Factory Method)
- ShapeCloner (Prototype)
- ComputerBuilder (Builder)

## Structural

- MediaPlayerAdapter (Adapter)
- CoffeeShop (Decorator)
- ImageLoader (Proxy)

## Behavioral

- WeatherStation (Observer)
- PaymentGateway (Strategy)
- RemoteControl (Command)
- VendingMachine (State)
- ReportGenerator (Template Method)
- SupportDesk (Chain of Responsibility)
