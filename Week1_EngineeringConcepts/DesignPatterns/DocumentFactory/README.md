# Factory Method Pattern Example

## Objective
Demonstrate the Factory Method Design Pattern using C#.

## Description
The Factory Method Pattern defines an interface for creating objects while allowing subclasses to decide which object to instantiate.

## Classes

- Document (Abstract Product)
- WordDocument (Concrete Product)
- PdfDocument (Concrete Product)
- DocumentFactory (Abstract Factory)
- WordFactory (Concrete Factory)
- PdfFactory (Concrete Factory)

## Output

```
Opening Word Document...
Opening PDF Document...
```

## Pattern Used

Creational Design Pattern
