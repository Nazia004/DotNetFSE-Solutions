# Exercise 6 – Library Management System

## Objective

Implement Linear Search and Binary Search to search books by title.

## Scenario

Develop a library management system that allows users to search for books efficiently.

## Algorithms Used

- Linear Search
- Binary Search

## Time Complexity

### Linear Search
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

### Binary Search
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)

## Conclusion

Linear Search works on both sorted and unsorted data, whereas Binary Search requires sorted data but performs much faster on large datasets.
