# Exercise 2 – E-commerce Platform Search Function

## Objective

Implement Linear Search and Binary Search algorithms for product search.

## Scenario

Develop the search functionality for an e-commerce platform where products can be searched efficiently.

## Algorithms Used

- Linear Search
- Binary Search

## Time Complexity

### Linear Search
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

### Binary Search
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)

## Conclusion

Binary Search is more efficient for large sorted datasets, whereas Linear Search is suitable for small or unsorted collections.
