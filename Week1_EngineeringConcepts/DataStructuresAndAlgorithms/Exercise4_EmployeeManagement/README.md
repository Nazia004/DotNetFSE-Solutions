# Exercise 4 – Employee Management System

## Objective

Implement an Employee Management System using an array.

## Operations

- Add Employee
- Search Employee
- Display Employee Records
- Delete Employee

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

## Advantages of Arrays

- Fast access using index
- Simple to implement
- Memory efficient for fixed-size data

## Limitations

- Fixed size
- Deletion requires shifting elements
- Insertion in the middle is expensive

## Conclusion

Arrays are suitable for applications with a fixed number of records but become less efficient when frequent insertions and deletions are required.
