# Exercise 1 - Inventory Management System

## Objective

Develop an Inventory Management System using C# to efficiently manage products in a warehouse.

## Features

- Add Product
- Update Product
- Delete Product
- Display Products

## Data Structure Used

- Dictionary<int, Product>

## Files

- Product.cs
- Inventory.cs
- Program.cs

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add Product | O(1) |
| Update Product | O(1) |
| Delete Product | O(1) |
| Search Product | O(1) |
| Display Products | O(n) |

## Technologies Used

- C#
- .NET Console Application
- Visual Studio Code
