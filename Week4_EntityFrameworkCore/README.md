# Week 4 - Entity Framework Core

## Overview

This week focuses on learning Entity Framework Core (EF Core), Microsoft's Object Relational Mapper (ORM) for .NET. The labs cover creating a DbContext, applying migrations, inserting data, and retrieving data from a SQL Server database using EF Core.

## Labs Completed

### Lab 1 - Understanding ORM
- Introduction to Object Relational Mapping (ORM)
- Understanding Entity Framework Core
- Creating entity classes

### Lab 2 - Setting Up DbContext
- Creating DbContext
- Configuring SQL Server connection
- Working with DbSet

### Lab 3 - Migrations
- Creating EF Core migrations
- Updating the database
- Understanding migration files

### Lab 4 - Seed Data
- Inserting records into the database
- Using SaveChanges()
- Working with entity relationships

### Lab 5 - Retrieving Data
- Retrieving records using Entity Framework Core
- Displaying database records
- Using DbSet to query data

## Technologies Used

- C#
- .NET
- Entity Framework Core
- SQL Server
- Visual Studio Code
